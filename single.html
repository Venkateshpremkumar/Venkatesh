Most impressive application of Computer Vision I have ever seen

So let’s start with what is computer vision. This technology enables the computer to see, understand and process images similar to humans with the help of certain image processing algorithms. For example, a vehicle can be equipped with a computer vision system to identify and differentiate the objects on its way to move accordingly.

Listed below are some of the popular computer vision application that is being researched extensively. 1. Surveillance robot: Making use of Raspberry Pi and Python Programming, this project helps you to develop a robot that possesses the quality of transmitting live video footage. 2. Surveillance Camera using IoT: This basically involves Image processing and its algorithms, Matpolib and splitting image channels. It revolves around IoT which is just another flawless technology. This surveillance camera using IoT that can be used to monitor your things for real-time safety. 3. Sixth Sense Robot: A robot built using Arduino as a microcontroller. An object tracking algorithm and the coloured object recognition algorithm is used to develop the robot. This can be used to classify harmful objects and that what you see in the film Robocop 4. Computer Vision-based Text Scanner: As the name suggests, a computer that can extract text from images or other 3D models and can display it on your screens. This involves optical character recognition, image thresholding, image processing algorithms, etc. 5. Computer Vision-Based Mouse: A very exciting application of CV that lets you perform various tasks of a computer mouse by just displaying corresponding colours in the webcam. Mainly image processing algorithms, image thresholding, canny edge detection, and object tracking are being used to achieve this. 6. Computer Vision-Based Smart Selfie: Wouldn’t it be cool if you could develop a computer vision-based smart selfie that can automatically take snaps when you smile? Definitely yes! And you see this lot in your smartphones nowadays. A smart selfie can be made by using image processing algorithms, face recognition, facial feature recognition, and Dlib C++ library.

What is transfer learning? How do you train a neural network by using it?

The best example of a transfer learning model in NLP is BERT. It's pre-trained on an extremely large corpus and we can just finetune it on smaller datasets. It's believed to produce a state-of-the-art results on various NLP tasks.
